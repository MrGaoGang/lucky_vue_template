
This is a webpack template for vue; the base config contains iview+vuex+vue-router

### Usage

``` bash
$ npm install -g vue-cli
$ vue init mrgaogang/lucky_vue_template my-project
$ cd my-project
$ npm install
$ npm run dev
```
